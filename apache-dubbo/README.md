# [Apache Dubbo](https://github.com/apache/incubator-dubbo)

## 分层架构
* 服务接口层（Service）
* 配置层（Config）
* 服务代理层（Proxy）
* 服务注册层（Registry）
* 集群层（Cluster）
* 监控层（Monitor）
* 信息交换层（Exchange）
* 网络传输层（Transport）
* 数据序列化层（Serialize）
