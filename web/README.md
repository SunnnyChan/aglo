CometD 框架是基于 HTTP 的事件驱动通信解决方案。第 2 版本中添加了对注释配置和 WebSocket 的支持。
CometD 的事件驱动方法非常适合事件驱动 Web 开发的新概念。

基于 Bayeux 标准化通信协议，Bayeux 通信协议主要是基于 HTTP。
它提供了客户端与服务器之间的响应性双向异步通信。
Bayeux 是一种 “发布- 订阅” 协议。
